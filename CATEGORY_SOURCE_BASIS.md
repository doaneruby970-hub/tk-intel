# Category Source Basis

This file explains what the Malaysia category radar is based on.

## Official layers

### TikTok Shop creator-facing top-level categories

Source:

- https://business.tiktokshop.com/us/creator/live?lang=en

Use in this project:

- Mapped the official high-level buckets such as `Fashion`, `Beauty and personal care`, `Tech and electronics`, `Foods and health`, `Toys and collectibles`, `Books and reading`, `Sports and outdoors`, `Automotive and motorcycle`, `Home improvement`, `Kitchenware`, `Furniture and decor`, and `Pet supplies`.

Limit:

- This is a public creator-facing top-level view, not a full authenticated seller leaf taxonomy.

## Malaysia broad-category layer

### FindNiche Malaysia top sellers page

Source:

Limit:

- This is a third-party public broad-category grouping, not TikTok's authenticated internal leaf export.

## Malaysia category-signal layer

### EchoTik SEA report

Source:

- https://static.echotik.live/report/tt_shop_q1.pdf

Use in this project:

- Used the report's Malaysia category signal to mark stronger categories such as `Beauty & Personal Care`, `Muslim Fashion`, `Food & Beverages`, `Textiles & Soft Furnishings`, and other categories appearing in MY top-10 GMV signal.

Limit:

- This is a market report and category signal layer, not a raw platform taxonomy.

### Bernama report on TikTok Shop Malaysia

Source:

- https://bernama.com/en/general/news.php?id=2413326

Use in this project:

- Used the article's mention of `Muslim fashion`, `beauty and personal care`, and `food and beverages` as top-selling Raya segments in Malaysia.

## What is curated by this project

The following parts are research-oriented and were created locally:

- subcategory splits
- micro-niche rows
- priority levels such as `core`, `watch`, `cautious`, `late`
- starter keyword suggestions
- content/logistics/compliance scores

These are selection heuristics, not official TikTok labels.
